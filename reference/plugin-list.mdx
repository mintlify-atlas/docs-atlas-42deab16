## Overview

This page lists all 42 plugins installed in nvim-luffy, organized by category. Plugin versions are locked in `lazy-lock.json` to ensure reproducible builds.

<Note>
  Versions shown are commit hashes from the lock file. Run `:Lazy update` to update plugins.
</Note>

## Managing Plugins

### Lazy.nvim Commands

<CodeGroup>
```vim Update All Plugins
:Lazy update
```

```vim Sync Plugins
:Lazy sync
```

```vim Check Plugin Health
:Lazy health
```

```vim View Plugin Status
:Lazy
```
</CodeGroup>

### Lock File

The `lazy-lock.json` file ensures all contributors use the same plugin versions:

<Info>
  Commit the lock file to version control to maintain consistency across environments.
</Info>

<Tip>
  To add a new plugin, create a file in `lua/leyland/plugins/` returning a lazy.nvim spec. The plugin will be automatically detected and installed.
</Tip>
