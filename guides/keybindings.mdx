## Overview

This is a comprehensive reference of all keybindings configured in nvim-luffy. The leader key is set to `<Space>` by default.

<Note>
  **Leader Key**: `<Space>`
  
  All keybindings shown with `<leader>` use the Space key by default.
</Note>

## Core Keybindings

These keybindings are defined in `lua/leyland/core/keymaps.lua`.

### Window Management

| Keybinding | Description |
|------------|-------------|
| `<leader>\|` | Split window vertically |
| `<leader>sh` | Split window horizontally |

### Buffer Management

| Keybinding | Description |
|------------|-------------|
| `<leader>bd` | Close current buffer |
| `<leader>ba` | Close all buffers |

## Telescope Keybindings

Telescope is the fuzzy finder. These keybindings are defined in `lua/leyland/plugins/telescope.lua`.

### Finding Files and Content

| Keybinding | Description |
|------------|-------------|
| `<leader><leader>` | Fuzzy find files in current working directory |
| `<leader>fr` | Find recent files (oldfiles) |
| `<leader>/` | Live grep - search for string in current working directory |
| `<leader>fc` | Find string under cursor in current working directory |
| `<leader>ft` | Find TODO comments |
| `<leader>fk` | Find keymaps |

### Telescope Insert Mode Mappings

When inside Telescope's search prompt:

| Keybinding | Description |
|------------|-------------|
| `<C-k>` | Move to previous result |
| `<C-j>` | Move to next result |
| `<C-q>` | Send selected items to quickfix list and open it |

## File Explorer (nvim-tree) Keybindings

File explorer keybindings. Defined in `lua/leyland/plugins/nvim-tree.lua`.

### Global Keybinding

| Keybinding | Description |
|------------|-------------|
| `<leader>e` | Toggle file explorer (finds and focuses current file) |

### Inside nvim-tree Window

These keybindings work when your cursor is in the nvim-tree window:

| Keybinding | Description |
|------------|-------------|
| `l` | Open file or expand directory |
| `h` | Close directory (go to parent) |

<Tip>
nvim-tree also includes many default keybindings. Press `g?` inside nvim-tree to see all available commands.
</Tip>

## LSP Keybindings

Language Server Protocol keybindings for code navigation and actions. Defined in `lua/leyland/lsp.lua`.

These keybindings are automatically set when an LSP server attaches to a buffer.

### Navigation

| Keybinding | Description |
|------------|-------------|
| `gR` | Show LSP references (via Telescope) |
| `gd` | Go to definition |
| `gi` | Show LSP implementations (via Telescope) |
| `gt` | Show LSP type definitions (via Telescope) |
| `K` | Show documentation for symbol under cursor |

### Code Actions

| Keybinding | Mode | Description |
|------------|------|-------------|
| `<leader>ca` | Normal, Visual | See available code actions |
| `<leader>rn` | Normal | Smart rename symbol |

### Diagnostics

| Keybinding | Description |
|------------|-------------|
| `<leader>d` | Show diagnostics for current line (floating window) |
| `<leader>D` | Show buffer diagnostics (via Telescope) |
| `[d` | Go to previous diagnostic |
| `]d` | Go to next diagnostic |
| `<leader>xx` | Toggle Trouble diagnostics panel |

### LSP Management

| Keybinding | Description |
|------------|-------------|
| `<leader>rs` | Restart LSP server |

## Flash Navigation

Flash provides enhanced motion commands. Defined in `lua/leyland/plugins/flash.lua`.

| Keybinding | Mode | Description |
|------------|------|-------------|
| `s` | Normal, Visual, Operator | Jump to any location with 2-character search |
| `S` | Normal, Visual, Operator | Jump using treesitter nodes |
| `r` | Operator | Remote flash for operator pending mode |
| `R` | Visual, Operator | Treesitter search |
| `<C-s>` | Command | Toggle flash search in command mode |
| `<C-Space>` | Normal, Visual, Operator | Incremental treesitter selection |

### Flash Character Actions

After triggering flash with `s`:

| Keybinding | Description |
|------------|-------------|
| `;` | Go to next match |
| `,` | Go to previous match |

## Git Integration

### Neogit

Defined in `lua/leyland/plugins/neogit.lua`.

| Keybinding | Description |
|------------|-------------|
| `<leader>gg` | Open Neogit (floating window) |

### Diffview

Defined in `lua/leyland/plugins/diffview.lua`.

| Keybinding | Description |
|------------|-------------|
| `<leader>dd` | Open diff view |
| `<leader>df` | Close diff view |
| `<leader>ds` | Open diff view comparing current branch to origin/main |

## Formatting

Defined in `lua/leyland/plugins/formatting.lua`.

| Keybinding | Mode | Description |
|------------|------|-------------|
| `<leader>mp` | Normal, Visual | Format file or selection |

<Note>
Formatting also runs automatically on save with a 3-second timeout.
</Note>

## Autocompletion (nvim-cmp)

These keybindings work in insert mode when the completion menu is active. Defined in `lua/leyland/plugins/nvim-cmp.lua`.

| Keybinding | Description |
|------------|-------------|
| `<C-k>` | Select previous suggestion |
| `<C-j>` | Select next suggestion |
| `<C-b>` | Scroll documentation up |
| `<C-f>` | Scroll documentation down |
| `<C-Space>` | Trigger completion suggestions |
| `<C-e>` | Close completion window |
| `<CR>` | Confirm selection (Enter) |
| `<Tab>` | Select next item or expand snippet |

## Default Vim Keybindings

nvim-luffy preserves most default Vim keybindings. Here are some essential ones:

<Tabs>
  <Tab title="Normal Mode">
    - `h/j/k/l` - Move left/down/up/right
    - `w/b` - Move word forward/backward
    - `gg/G` - Go to top/bottom of file
    - `0/$` - Go to start/end of line
    - `o/O` - Insert new line below/above
    - `dd` - Delete line
    - `yy` - Copy line
    - `p/P` - Paste after/before cursor
    - `u` - Undo
    - `<C-r>` - Redo
  </Tab>

  <Tab title="Visual Mode">
    - `v` - Character-wise visual mode
    - `V` - Line-wise visual mode
    - `<C-v>` - Block-wise visual mode
    - `y` - Yank (copy) selection
    - `d` - Delete selection
    - `>/<` - Indent/unindent
  </Tab>

  <Tab title="Insert Mode">
    - `<Esc>` - Return to normal mode
    - `<C-w>` - Delete word backward
    - `<C-h>` - Delete character backward
  </Tab>
</Tabs>

<Info>
For a complete list of all keybindings, including those from plugins, run `:Telescope keymaps` in Neovim.
</Info>
